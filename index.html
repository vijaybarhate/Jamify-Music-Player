About JAMIFY

JAMIFY is a music streaming web application designed to replicate a simplified version of popular music streaming platforms.

Features:

  • Play, pause, and skip through a list of preloaded songs.
  • Modern component-based UI with hover effects.
  • Responsive design for both desktop and mobile devices.
  • Search filter for quick track discovery.
